    # git 多人合作

    在组织上创建一个仓库存放项目，并提交项目到本仓库，其他成员就可以克隆项目（克隆下来的项目不需要 git init 初始化了）

    # 克隆指令

    git clone 仓库地址

    # 上传代码

    1. 查看被修改的文件
    git status
    1. 选择所有未提交的文件
    git add .
    1. 添加注释
    git commit -m'注释'
    1. 推送到自己的远程分支
    git push origin 自己的分支名
    1. 将分支切换到主分支
    git checkout 主分支名
    1. 主分支合并自己的分支
    git merge 自己的分支名
    1. 拉取代码
    git pull

    # *如果没有冲突直接执行

    git push origin 主分支名

    # *如果有冲突

    git add .
git commit -m'注释'
git push origin 主分支名
    1. 切换到自己的分支
    git checkout 自己的分支名

    # 当你需要小组其他成员的代码时需要下拉代码

    # 下拉代码

    1. 将分支切换到主分支
    git checkout 主分支名
    1. 拉取代码
    git pull
    1. 切换到自己的分支
    git checkout 自己的分支名
    1. 自己的分支合并主分支
    git merge 主分支名

    # 强制退出并保存

    输入esc，和:wq
